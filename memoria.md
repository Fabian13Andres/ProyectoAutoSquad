## AutoSquad

**AutoSquad** es una aplicación multiplataforma diseñada para facilitar la conexión entre jugadores que desean formar equipos de manera rápida.

Surge para resolver un problema común: muchos usuarios quieren jugar en grupo pero no encuentran compañeros en el momento, lo que provoca pérdida de tiempo y el uso de múltiples plataformas externas poco organizadas.

### Objetivo
Centralizar todo el proceso en una única plataforma donde los usuarios puedan:
- Registrarse
- Crear salas de búsqueda de jugadores (“cajas”)
- Comunicarse mediante un sistema de mensajería integrado

De esta forma, se reduce el tiempo de búsqueda y se mejora la experiencia al ofrecer un entorno unificado.

### Arquitectura
AutoSquad está diseñada como una solución escalable y accesible desde distintos dispositivos (web, móvil y escritorio), basada en:
- Arquitectura cliente-servidor  
- API REST  
- Base de datos centralizada  

Esto permite separar responsabilidades, facilitar el mantenimiento y posibilitar futuras ampliaciones del sistema.
