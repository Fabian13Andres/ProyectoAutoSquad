# AutoSquad.md

## Estructura del Proyecto

En este proyecto disponemos de **4 carpetas principales**, organizadas de la siguiente manera:
- `autosquad-api`: Contiene la API del sistema.
- `autosquad-web`: Aplicación web.
- `AutoSquadDesktop`: Aplicación de escritorio.
- `AutoSquadForAndroid`: Aplicación móvil para Android.

## Documentación y Recursos

Además del código fuente, el proyecto incluye:
- Un documento PDF con la **documentación técnica** en español y otro idéntico en inglés.
- Imágenes del **logo de la aplicación**.
- Un archivo `.graphml` (compatible con yEd) que representa un **esquema del objetivo y estructura futura de la app**.

## Archivos Adicionales

También se incluyen algunos archivos `.txt` con información útil:
- Base de datos SQL.
- Guías básicas con comandos y recordatorios usados en el proyecto.

## Objetivo

El archivo de esquema en yEd refleja la idea general de lo que se quiere conseguir con la aplicación AutoSquad a futuro, sirviendo como referencia visual para la evolución del proyecto.

---
*Este repositorio está pensado como una base organizada para el desarrollo multiplataforma de AutoSquad.*
